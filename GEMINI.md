# GEMINI.md

This is a **fork of Obsidian Web Clipper** in which we are adding new features (live-webpage
annotation: highlights, comments, freehand drawing; a highlights dashboard; and sync).

## Where to read
- **Architecture & all features (implemented)** → `AGENTS.md`

## How to work here
- `AGENTS.md` is the single source of truth for what this fork does. Whenever a feature is
  **implemented or modified** in a way that changes behavior worth knowing about, record it in
  `AGENTS.md`.
- Keep those notes at a **high / mid level** — describe the feature and any necessary detail to
  understand it, but do **not** drop into code-level depth or log individual bug fixes.
- Keep `AGENTS.md` concise but comprehensive.
